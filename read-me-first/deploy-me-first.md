# Deploy Me First

This folder contains the cumulative WorkRightsUK build up to Batch 20.

Recommended use:
- deploy from the full Batch 20 checkpoint zip
- keep the main reference zip untouched as your clean backup
- use the incremental Batch 20 zip only for comparing what changed in this batch

Suggested checks after deployment:
- homepage loads correctly
- tools index, guides index, help index, compare index load
- redundancy, holiday, settlement, paternity, and maternity tools load
- methodology, sources, contact, and key help pages load
- sitemap.xml is present and reachable
