# Batch 18 SEO notes

## Main search-facing patterns targeted
- Google title-link guidance stresses making the main title clear and prominent
- Google snippet guidance stresses writing page-specific descriptions that accurately summarise the page
- Google’s people-first guidance still favours clear page purpose and satisfying user experience over search-first padding

## What Batch 18 changed because of that
- Top page titles are now more explicit about UK scope, calculator/guide type, and next-step value
- Meta descriptions are more specific to each page rather than generic
- H1s are tighter and clearer on the main index and tool pages
- Internal anchors are more descriptive, helping both users and search engines understand the destination better
