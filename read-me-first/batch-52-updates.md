# Batch 52 updates

This batch focuses on the help hub and internal-linking quality.

## What changed
- Reworked the help hub so it behaves more like a curated next-step area and less like a loose list of pages.
- Added quick links at the top of the help hub for final pay, redundancy next steps, settlement checks, and family-rights next steps.
- Reordered the help hub so the strongest entry points appear first: compare, final pay, redundancy next steps, settlement checks, then family-rights next steps.
- Expanded help-hub ItemList schema to include the family-rights next-step page.
- Added a second support section on the help hub linking out to Compare, Tools, Contact, and About.
- Replaced "Live" badges on the Tools and Guides hubs with steadier labels like "Core tool" and "Core guide".
- Added clearer cross-links between the Tools, Guides, Help, and Compare hubs so users can move between sections more naturally.
- Improved the Compare hub by replacing a duplicate family comparison card with the more useful family-rights next-step help page.
- Added one extra link from the final-pay help page back into the wider help area and settlement-check route.

## Why this batch matters
- The site already has the right pages, but the support area needed to feel more curated.
- This batch makes it easier to move from calculators to help pages and from comparison pages to the right next action.
- It also removes a few remaining dashboard-style labels that made hub pages feel less stable.
