How WorkRightsUK builds pages

This page explains the method behind WorkRightsUK so users can understand how calculators, guides, and comparison pages are put together.

This page exists to make the site’s source use, scope limits, and practical design choices easier to verify.

Core approach

Start with official GOV.UK and Acas guidance, then build a clearer user-facing page around the parts most people actually need first.

Scope limit

WorkRightsUK is not a substitute for personalised legal or tax advice on disputed facts, deadlines, or complex case-specific issues.

1. Official source first

Pages are built around GOV.UK and Acas where applicable, with source links shown on the page so users can verify the rule basis.

2. Narrow practical scope

Each page focuses on the most useful first-check answer rather than trying to solve every legal scenario at once.

3. Clear next-step routing

Where a number alone is not enough, pages should point users to the next most useful page on the site.

4. Reviewed-date visibility

Pages are designed to show reviewed dates and rule-version awareness so the user can see that timing matters.
