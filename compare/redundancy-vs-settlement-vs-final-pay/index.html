  2. Use final pay next if notice, holiday, wages or PILON are part of the package.

Best first page by situation

Common mixed situations

  • Redundancy + notice + holiday: start with redundancy, then check final pay.
  • Settlement offer + PILON + holiday pay: start with settlement, then use final pay.
  • Dismissal feels unfair: start with unfair dismissal before you focus only on the payment lines.
  • Last payslip looks wrong: start with final pay even if redundancy or settlement is also involved.
If the payslip is the problem, start with final pay first

Even if redundancy or settlement is also involved, use the final-pay page first when the immediate issue is what should appear in the last payment, what is missing, or whether deductions look wrong.
