Common questions

Source list FAQs

Why does WorkRightsUK link official sources?

WorkRightsUK links directly to GOV.UK, Acas, HMRC and other primary sources so users can verify the rule basis behind a calculator, guide or help page.

Does this page list every employment-law source someone might need?

No. It lists the main official sources currently used across the core pages, not every possible tribunal, case-law or specialist reference.

When should I recheck an official source?

Recheck when rates, caps, dates, tax treatment or eligibility rules could have changed, or when your situation is disputed or deadline-sensitive.
