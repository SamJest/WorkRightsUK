Why sources are shown

WorkRightsUK is designed to be practical and verifiable, not just persuasive. Users should be able to open the official source links directly.

See how pages are checked

What this page covers

The core official sources currently in use and why they are tracked.

What this page does not cover

Every possible employment-law source or every source that may be added later as the site grows.

How to use this page

Start from a tool or guide first, then use this page to verify the official source basis behind the strongest calculators and help pages.

Check the date

The source rows show when the entry was last checked, not when the underlying rule first came into force.

Check the page type

The site prioritises GOV.UK, Acas and HMRC pages that users can open directly.

Check the fit

Open the source that matches your real issue before relying on a figure, especially where dates, pay labels or dismissal reasons matter.
