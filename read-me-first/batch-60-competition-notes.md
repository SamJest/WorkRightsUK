# Batch 60 - Competition notes

## Why this batch
The holiday page already had sensible guardrails, but it still behaved more like one calculator with caveats than a route chooser.

## Competitor gap being closed
- GOV.UK is stronger at branching users into the right holiday route, especially when irregular-hours or part-year cases appear.
- CharlieHR is stronger at making standard, starter and leaver choices feel immediate before calculation begins.

## WorkRightsUK advantage targeted here
Keep the stronger consumer-facing “what this means / what next” framing, but make the route choice faster so users know whether to:
- stay on the regular-pattern calculator
- use starter mode
- use leaver mode
- switch to GOV.UK instead

## What to review after deploy
- whether mode choice feels obvious above the fold
- whether the mode buttons help mobile users start faster
- whether the result now feels more clearly tied to the selected mode
